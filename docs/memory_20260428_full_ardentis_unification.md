# Memory: ARDENTIS LEADS - Unificação de Design - 2026-04-28

## Status
- Versão: 5.0 (Full Ardentis Suite)
- Build: Sucesso (npm run build)

## Unificação de Páginas
- **Clientes**: Reestruturada com cards `glass`, métricas de uplink e botões de ação modernos.
- **Webhooks**: Implementação de um terminal de resposta com sintaxe destacada e formulários integrados ao design DeFi.
- **Dashboard do Cliente**: Replicado o estilo de "Overview" para visualização individual de entidades, com logs de transmissão e mapeamento de dados dinâmicos.

## Refinamentos Visuais & Fontes
- **Tipografia**: Otimização da fonte **Outfit** com ajustes de peso (700 para títulos, 500 para corpo) e espaçamento entre letras (-0.02em para títulos).
- **Consistência**: Todos os componentes internos agora utilizam as variáveis globais Ardentis, garantindo que o fundo granulado e as cores neon sejam onipresentes.
- **Interatividade**: Botões pill-shaped com sombras de elevação ciano e estados de hover refinados.

## Melhorias Técnicas
- Padronização do uso de Client Components onde há necessidade de hooks de navegação ou eventos.
- Limpeza de estilos residuais do layout antigo (Jarvis).
